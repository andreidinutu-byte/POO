Proiect facut pentru materia Programare Orientata pe Obiecte in cadrul facultatii.
Proiectul a fost conceput pentru CLION, deci trebuie folosit si fisierul CMakeLists pentru a-l putea rula.
Este un proiect umoristic, un joc cu gladiatori, cu o harta pe care te poti plimba cu personajul tau, avand si un sistem avansat de gestionare a gladiatorilor
Nu este finalizat, este adus doar pana in punctul in care sa respecte toate cerintele de la facultate, urmand sa fie reluat si terminat pentru a fi mai interactiv.
